# node-crud

```js

npm init -y
```

instalando o nodemon
```js
npm i nodemon
```
no package.json script

```json
"start": "nodemon server.js",
```


adicionar o module após o "main"

```json
"type": "module"
```

criar o .gitignore e adicionar a pasta *node_modules* para não ser enviado ao repositorio remoto

```
node_modules
```
